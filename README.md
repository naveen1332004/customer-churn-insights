# Customer Churn Insights

**Overview**  
Conducted end-to-end analytics to identify drivers of customer churn and deliver retention recommendations.

**Key contributions**
- Performed SQL & Python-based exploratory data analysis and feature engineering to surface churn risk drivers.
- Built interactive dashboards (Tableau / Power BI) to visualize churn patterns, cohort trends, and KPIs.
- Developed predictive models and improved churn prediction error by **20%**, enabling targeted retention strategies.

**Project structure**

**How to run**
1. Clone the repo: `git clone https://github.com/naveen1332004/customer-churn-insights`
2. Create venv and install: `python -m venv venv && source venv/bin/activate && pip install -r requirements.txt`
3. Open `/notebooks/` for EDA and modeling; dashboards are in `/dashboards/`.

**Highlights**
- Tools: Python (pandas, scikit-learn), SQL, Tableau, Power BI
- Impact: Reduced churn prediction error by 20% and generated actionable retention playbooks
